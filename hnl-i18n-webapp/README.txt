Sample Message ResourceBundle extension hnl-i18n-webapp to show how you can override the ResourceBundle class to your own with some addition features to sequence the locale in order if a key for a locale does not exist.

============================
1.  Install Maven and Java
2.  On the Root Directory run  "mvn clean install"
3.  Then run "mvn tomcat:run-war" to startup the server
4.  You can access the webapp with:  

http://localhost:8080/hnl-i18n-webapp
